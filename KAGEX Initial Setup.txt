Title: KAGEX Initial Setup Method

Date: 2012/4/29

Author: Go Watanabe



1. Obtaining the Base



　Use Subversion to obtain the following from the official Kirikiri repository:



　  https://sv.kikyou.info/svn/kirikiri2/tags/kag3ex3



2. Deploying Executable Files



  Download the Kirikiri binary distribution from the repository and place it here:

  https://sv.kikyou.info/svn/kirikiri2/trunk/kirikiri2/bin/win32



  *Note: Operation has only been confirmed with the latest development binary (Rev.5034) or later.



 Executing krkr.exe will start the sample system.



3. About Sample Data



  template/   These are the base template files.

  data/       This is a sample that includes system scripts (title screen, etc.).



  Files under data/ reference template/system.



  When actually releasing a game, it is better to copy template/system and template/startup.tjs 

　directly into the data directory.
